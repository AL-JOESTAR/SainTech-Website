### Project Belajar Pemrograman Web

Project ini merupakan website Company Profile yang dibuat sebagai sarana belajar pemrograman web. Fokus utama dari project ini adalah membuat website yang dinamis, di mana seluruh konten dapat diedit melalui admin panel tanpa perlu mengubah kode secara langsung.

Dengan adanya sistem admin panel, proses pengelolaan dan update isi website menjadi lebih mudah, cepat, dan efisien.
